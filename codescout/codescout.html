Documentation de l'API RESTful de Codescout

Introduction

Bienvenue dans l'API RESTful de Codescout! Cette API permet de générer des images contenant des messages codés à l'aide de divers codes scouts. Vous pouvez utiliser les endpoints pour créer et récupérer ces images à partir de requêtes HTTP.

URL de base

Toutes les requêtes API doivent être faites à la base URL suivante:

https://scouts.pythonanywhere.com/

Endpoints

URL: /codescout

Méthode: GET

Description: Ce endpoint permet de générer une image avec un message codé selon le code scout spécifié.

Paramètres

Paramètre Type Requis Défaut Description Exemple
message string Oui Le message à encoder message=Toujours%Prêt
code string Oui Le code à utiliser pour encoder le message code=musical
decalage integer Non Décalage à utiliser pour le code alphabet decalage=8
delimiteur string Non : Caractère utilisé pour diviser le message en plusieurs lignes delimiteur=:
taille integer Non 8 Taille des éléments du message en pixels taille=10
interligne integer Non 1 Espace entre les lignes de code pour faciliter le décodage interligne=2
bordure integer Non 1 Taille de la bordure en pixels bordure=2
legende string Non Légende ajoutée en bas à gauche de l'image legende=Exemple
fonte string Non DejaVuSansMono.ttf Nom de la police à utiliser pour la légende et le texte décodé fonte=OpenSans-Regular.ttf
decoder boolean Non 1 Indique s'il faut afficher le message décodé (1 ou 0) decoder=1

Codes disponibles

  • code soleil: chaque lettre est remplacée par un soleil dont le nombre de rayons indique la position dans l'alphabet
  • code musical: chaque lettre est remplacée par une note de musique où les lettres de A-M deviennent des blanches entre DO central et LA aigu et les lettres de N-Z deviennent des noires entre DO central et LA aigu
  • code alphabet: chaque lettre est décalée par un nombre de lettres dans l'alphabet défini par le paramètre "décalage"
  • code avocat: équivalent au code alphabet avec un décalage de +8
  • code escalier: équivalent au code alphabet avec un décalage de -10

Exemple de Requête

GET https://scouts.pythonanywhere.com/codescout?message=De%20notre%20mieux&code=musical&taille=10&bordure=2&legende=Légende&decoder=1&interligne=3

Réponse

Mime Type: image/png

Description: Une image PNG contenant le message encodé selon le code scout spécifié.

Exemples

Générer une image simple:

GET https://scouts.pythonanywhere.com/codescout?message=Bonjour&code=soleil

Ajouter des options de personnalisation:

GET https://scouts.pythonanywhere.com/codescout?message=Toujours%20Prêt&code=musical&taille=12&bordure=3&legende=Scouts%20du%20Canada&decoder=1&interligne=3

Encoder "Baden Powell" sur deux lignes avec le code musical:

GET https://scouts.pythonanywhere.com/codescout?message=Baden:Powell&code=musical

Encoder "Baden Powell" sur deux lignes avec le code soleil en augmentant la taille et en changeant la fonte:

GET https://scouts.pythonanywhere.com/codescout?message=Baden:Powell&code=soleil&taille=15&fonte=Cousine-Regular.ttf

Encoder "Justice Respect Confiance Dépassement" sur quatre lignes avec le code soleil, ne pas afficher le message décodé et augmenter l'espace entre les lignes:

GET https://scouts.pythonanywhere.com/codescout?message=Justice:Respect:Confiance:Dépassement&code=soleil&decoder=0&interligne=3
