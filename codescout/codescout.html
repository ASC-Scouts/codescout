Interface RESTful pour le package python codescout

Cet API RESTful permet de générer des messages à décoder en utilisant différentes méthodes d'encodage populaires chez les Scouts.

Voir https://github.com/ASC-Scouts/codescout

Paramètres de l'API:

Paramètre Valeur
message message a encoder
code code à utiliser, parmi les codes suivants: soleil, musical
taille taille des elements du message en pixels [défaut 8]
delimiteur caractere a utiliser dans le message pour indiquer a l'API de decouper le message sur plusieurs lignes a l'aide de ce delimiteur [défaut :]
interligne permet d'ajouter de l'espace entre les lignes du message afin par exemple de laisser de la place pour decoder [défaut 1]
bordure permet d'ajouter une bordure au message afin par exemple de faciliter son decoupage (le paramètre correspond a la taille en pixels de la bordure) [défaut 0]
legende permet d'ajouter une legende en bas a gauche (par exemple une courte instruction, un numéro...) [défaut aucune]
decoder indique si oui ou non on doit afficher le message décodé sous le code (1=oui, 0=non) [défaut 1]
fonte nom de la fonte a utiliser pour la legende et le texte décodé [défaut DejaVuSansMono.ttf]

Exemples d'utilisation:

Encoder "Baden Powell" sur deux lignes avec le code musical:

https://scouts.pythonanywhere.com/codescout?message=Baden:Powell&code=musical

Encoder "Baden Powell" sur deux lignes avec le code soleil en augmentant la taille et en changeant la fonte:

https://scouts.pythonanywhere.com/codescout?message=Baden:Powell&code=soleil&taille=15&fonte=Cousine-Regular.ttf

Encoder "Justice Respect Confiance Dépassement" sur quatre lignes avec le code soleil, ne pas afficher le message décodé et augmenter l'espace entre les lignes:

https://scouts.pythonanywhere.com/codescout?message=Justice:Respect:Confiance:Dépassement&code=soleil&decoder=0&interligne=3
